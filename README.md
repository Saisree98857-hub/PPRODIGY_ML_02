# Customer Segmentation using K-Means

## 📌 Project Description
This project uses K-Means clustering to group customers based on:
- Annual Income
- Spending Score

## ⚙️ Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 📊 Output
- Elbow Method graph
- Customer segmentation clusters

## 🚀 How to Run
1. Install libraries:
   pip install numpy pandas matplotlib scikit-learn
2. Run:
   python kmeans.py
